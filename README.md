## セットアップ
rustupをインストールしてrustの開発環境を構築する場合
```bash
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh

# rustをアンインストールする場合
# rustup self uninstall
```

新規プロジェクトの作成  
```bash
cargo new #project_name
# Hello World を出力できる最小のプロジェクトが作成される。
```
